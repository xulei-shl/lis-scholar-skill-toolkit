# 日报模板与输出格式参考

## 日报模板（三章节结构）

```markdown
# Google Scholar 日报 - YYYY-MM-DD

## 统计摘要

| 指标 | 数量 |
|------|------|
| 处理邮件 | X 封 |
| 解析论文 | Y 篇 |
| 相关论文 | Z 篇 |

## 汇总摘要

[150-300字的综合性摘要]

本次推送共发现 Z 篇相关论文，主要集中在以下研究方向：

1. **主题一**：[简要描述]
2. **主题二**：[简要描述]

特别值得关注的是 [某个研究方向或论文]，这对图书馆学/信息资源组织领域具有重要参考价值...

---

## 相关论文详情

### 1. [论文标题] [PDF] [HTML]

- **作者**: Author1, Author2
- **来源**: Source, Year
- **相关度**: ★★★★★
- **匹配原因**: [自然语言描述论文与研究兴趣的关联]

**摘要片段**:
论文摘要片段...

---

### 2. [论文标题]

- **作者**: ...
...
```

## 论文详情条目格式

```markdown
### {序号}. {论文标题}{格式标记}

- **中文题名**: {原英文题名翻译成中文题名}
- **作者**: {作者列表}
- **来源**: {来源期刊}
- **相关度**: {星级评分}
- **匹配原因**: {匹配原因描述}

**摘要片段**:
{摘要片段}

---
```

### 格式标记说明

| 标记 | 条件 |
|------|------|
| `[PDF]` | `has_pdf == true` |
| `[HTML]` | `has_html == true` |

### 星级评分标准

| 星级 | 相关度 | 说明 |
|------|--------|------|
| ★★★★★ | 高度相关 | 直接针对图书馆/信息组织领域 |
| ★★★★☆ | 较为相关 | 有明确的应用场景 |
| ★★★☆☆ | 中等相关 | 可能有参考价值 |
| ★★☆☆☆ | 弱相关 | 仅提及相关概念 |
| ★☆☆☆☆ | 低相关 | 建议排除 |

## 汇总摘要生成指南

汇总摘要应包含：

1. **统计概述**：相关论文总数
2. **主要主题**：2-4 个主要研究方向
3. **特别关注**：值得深入阅读的论文或方向
4. **领域价值**：对图书馆学/信息资源组织领域的参考价值

### 示例

```markdown
本次推送共发现 6 篇相关论文，主要集中在以下研究方向：

1. **大语言模型在知识组织中的应用**：探索 LLM 时代的知识组织系统演进、元数据标准更新
2. **RAG 技术在数字图书馆中的应用**：检索增强生成技术在文献检索、知识问答中的实践
3. **知识图谱构建方法**：面向图书馆馆藏资源的知识抽取与图谱构建技术

特别值得关注的是《Knowledge Organization in the Era of Large Language Models》一文，该文系统分析了 LLM 对传统知识组织系统的冲击与机遇，对当前图书馆数字化转型具有重要参考价值。
```

## 空日报模板（无相关论文）

```markdown
# Google Scholar 日报 - YYYY-MM-DD

## 统计摘要

| 指标 | 数量 |
|------|------|
| 处理邮件 | X 封 |
| 解析论文 | Y 篇 |
| 相关论文 | 0 篇 |

## 汇总摘要

今日处理的 {Y} 篇论文中，没有发现与您研究兴趣高度相关的论文。

论文主题主要集中在 [简要分析今日论文主题分布]，这些主题与您当前关注的知识组织、元数据、大模型应用等方向关联度较低。

---

## 相关论文详情

今日无相关论文。
```

## Subagent 返回格式

每个 `scholar-email-processor` Subagent 应返回：

```json
{
  "email_id": "xxx",
  "email_subject": "Google Scholar Alert: ...",
  "total_papers": 10,
  "relevant_papers": [
    {
      "title": "论文标题",
      "authors": "作者列表",
      "source": "来源期刊",
      "snippet": "摘要片段",
      "has_pdf": true,
      "has_html": false,
      "relevance_score": "★★★★☆",
      "match_reasons": "自然语言描述匹配原因",
      "discipline_match": true
    }
  ],
  "filtered_count": 7
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `email_id` | string | Gmail 邮件 ID |
| `email_subject` | string | 邮件主题 |
| `total_papers` | number | 邮件中论文总数 |
| `relevant_papers` | array | 相关论文列表 |
| `filtered_count` | number | 被过滤掉的论文数 |

### relevant_papers 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | string | 论文标题 |
| `authors` | string | 作者列表（逗号分隔） |
| `source` | string | 来源期刊/会议 |
| `snippet` | string | 摘要片段 |
| `has_pdf` | boolean | 是否有 PDF 链接 |
| `has_html` | boolean | 是否有 HTML 链接 |
| `relevance_score` | string | 星级评分（★☆☆☆☆ 到 ★★★★★） |
| `match_reasons` | string | 匹配原因描述 |
| `discipline_match` | boolean | 是否匹配学科领域 |
