新建文件（夹）
标签：文件

请求说明
字段	值
请求地址
https://openapi.wps.cn/v7/drives/{drive_id}/files/{parent_id}/create
HTTP 方法
POST
接口描述
新建文件（夹）
签名方式
KSO-1
限频策略
无
权限要求
查询和管理文件(应用授权) kso.file.readwrite
查询和管理文件(用户授权) kso.file.readwrite
MCP开放能力(用户授权) kso.mcp.readwrite
请求头 (Headers)
属性名	类型	是否必填	描述	可选值
X-Kso-Id-Type
string	否	类型
* internal - 内部
* external - 外部
internal, external
路径参数 (Path)
属性名	类型	是否必填	描述	可选值
drive_id
string	是	驱动盘id	-
parent_id
string	是	文件夹id（根目录时为0）	-
请求体(Body)
请求体格式: application/json
属性名
类型	必填	描述
file_id	string	否	
快捷方式的源文件，只有在file_type = shortcut的时候需要

file_type	string	是	
folder - 文件夹
file - 文件
shortcut - 快捷方式
枚举值：
folder
file
shortcut
name	string	是	
文件名，须带上后缀，例: doc.docx(普通文件), abc.docx.link(快捷方式)，支持格式：doc,docx,form,xls,otl,ppt,dbt,xlsx,pptx，仅公网支持格式：pom,spt,dppt,link,resh,ckt,ddoc,dpdf,dxls,pof,wpsnote 仅私网支持格式：txt,ksheet

on_name_conflict	string	否	
新建时文件名冲突行为处理方式,不传为rename默认值。(该接口只识别rename和fail)

枚举值：
fail
rename
overwrite
replace
parent_path	array[string]	否	
该参数用于指定相对于当前文件目录的相对路径。数组中的每个元素代表一个路径名，而非路径ID。若指定的路径不存在，系统将自动创建该路径


请求体示例

{
  "file_id": "string",
  "file_type": "folder",
  "name": "string",
  "on_name_conflict": "fail",
  "parent_path": [
    "string"
  ]
}
