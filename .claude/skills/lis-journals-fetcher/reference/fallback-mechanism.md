# 回退机制详细说明

当目标年期爬取不到（空结果）时，严格按照以下顺序进行回退。

## 回退顺序

### 1. 同年递减期号
```
2025-06 → 2025-05 → 2025-04 → ... → 2025-01
```

### 2. 跨年回退
```
2025-01 → 2024-12 → 2024-11 → ...
```

### 3. 放弃回退
提示手动输入年期或跳过

## ⚠️ 重要约束

**严禁跨年跳过！** 例如：2025-06 不存在时，必须先尝试 2025-05，不能直接跳到 2024-12。

## 自动回退策略

- 最多自动回退 **3 期**
- 3 期均失败后，使用 `AskUserQuestion` 提示用户选择：
  - 手动输入年期
  - 跳过此期
  - 终止任务

## 示例流程

```python
# 目标：2025-06
result = fetch(2025-06)  # 空结果
result = fetch(2025-05)  # 空结果
result = fetch(2025-04)  # 成功 ✓
# 或
result = fetch(2025-01)  # 空结果
result = fetch(2024-12)  # 成功 ✓
```
